# Tweaker <sup><sub>Trackmania ManiaPlanet Turbo</sub></sup>
Plugin for OpenPlanet that helps modify in-game hidden settings and slightly boost the performance.

## Features
* Graphics tweaking
    * Screen Resolution
    * Draw Distance (Z-Clip)
    * Level of Detail (LoD)
    * Render Modes & Types
    * Shadow Types
    * Projectors
* Environment
    * Background
    * Clouds
    * Lighting
* Camera tweaking
    * Field of View
    * Quick Zoom
