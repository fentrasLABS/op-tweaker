# Tweaker <sup><sub>Trackmania ManiaPlanet Turbo</sub></sup>
Plugin for OpenPlanet that helps modify in-game hidden settings and slightly boost the performance.

## Features
* Graphics
    * Screen Resolution
    * Draw Distance (Z-Clip)
    * Level of Detail (LoD)
    * Render Modes
    * Lighting Modes
    * Projectors
* Environment
    * Background
    * Clouds
    * Lighting
* Camera
    * Field of View
* Interface
    * FPS counter
* Special
    * Quick Zoom
    * Wipeout Mode
